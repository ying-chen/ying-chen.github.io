Citation with Natbib package

Win10+Texlive+Texmaker


Finally get a satisfactory way to generate elegant citation in term paper :-)

Template

Choice on\bibliographystyle{apa}

  • I use {newapa}, the APA style, seems the best alternative plan for Harvard style.
  • Other choices: apa, agsm, apalike, astron, authordate1, authordate2, authordate3, authordate4, bbs, cbe, chicago, chicagoa, dcu, humanbio, humannat, jas99, jmb, jtb, kluwer, named, phppcf, plainnat

Changes on parameter of \setcitestyle

  • Citation mode: authoryear , number , super
  • Brackets: round , square, and any other format with open={char}, close={char}

Citation commands in text

Command Example Description
\citet{Asp49,And80} Aspin (1949); Andrews et al. (1980)
\citep{Asp49,And80} (Aspin 1949; Andrews et al. 1980)
\citeauthor{And80} Andrews et al.
\citeyear{And80} 1980
\cite[see][chap 1]{And80} (see Andrews et al., 1980, chap 1 ) faefv

Some notes in ‘Options’

  • Options –> Commands –> Bib(la)tex –> bibtex % , in order to avoid ERROR - Cannot find control file 'thesis-ref.bcf'! due to Natbib ; while sometimes should use biber % instead.
  • Options –> Quick Build –> Quick Build Command –> PdfLaTex + Bib(la)tex + PdfLaTex*2 + View PdF
